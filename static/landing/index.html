无与伦比的写作体验

丰富的 Markdown 编辑器

GitPen 的 Markdown 语法高亮是独一无二的。 编辑器的精致文本格式可帮助您可视化文件的最终呈现。

所见即所得控件

GitPen 提供了非常方便的格式化按钮和快捷方式，这要归功于 Stack Overflow 使用的所见即所得式 Markdown 编辑器 PageDown。

智能布局

无论你是写作、阅读还是评论……GitPen 的布局都为你提供了所需的灵活性。

滚动同步实时预览

GitPen 的滚动同步功能精确地绑定了编辑器面板和预览面板的滚动条，以确保您在编写时始终关注输出。

专为网络写手设计

保持同步

GitPen 可以将您的文件与 GitHub 和 Gitee 同步。 您可以选择是以 Markdown 格式、HTML 上传，还是使用 Handlebars 模板引擎格式化输出。

协作

借助 GitPen，您可以共享协作文档空间，这要归功于同步机制。 如果两个协作者同时处理同一个文件，GitPen 会负责合并更改。

评论

GitPen 允许您在文件中插入内联评论和嵌入协作者讨论，就像 Microsoft Word 和 Google Docs 一样。

离线写作！

即使在旅行时，GitPen 仍然可以访问，让您可以像任何桌面应用程序一样离线编写。 你没有借口再偷懒！

扩展的 Markdown 支持


GitHub 风格的 Markdown

GitPen 支持不同的 Markdown 风格，例如 Markdown Extra、GFM 和 CommonMark。 每个 Markdown 功能都可以在您方便的时候启用或禁用。


LaTeX 数学表达式

GitPen 从您的 Markdown 文件中的 LaTeX 表达式呈现数学公式。

UML 图

GitPen 使您能够使用简单的语法编写序列图和流程图。

乐谱

GitPen 可以使用 ABC 表示法渲染乐谱。

Emojis表情

GitPen 支持使用 Markdown 表情符号标记在文件中插入表情符号。
